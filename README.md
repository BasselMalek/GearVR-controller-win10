# GearVR-controller-win10

My personal take on interfacing with the Samsung GearVR Contoller on PC.

## Features

- [x] CLI interface.
- [x] Remappable buttons.
- [x] Touchpad tracking.
- [x] Touchpad D-pad.
- [x] 3d cursor (experimental, needs further calibration to fix drift/jitter).
